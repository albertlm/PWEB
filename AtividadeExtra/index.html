Ubuntu é uma distribuição Linux produzida pela empresa africana Canonical. Ele é um sistema operacional completo que pode ser instalado em computadores PC e Mac. O sistema é um dos Linux mais populares da atualidade.

Debian, anteriormente chamado de Debian GNU/Linux e hoje apenas de Debian é um sistema operacional composto inteiramente de software livre. O projeto recebe ainda apoio de outros indivíduos e organizações em todo mundo.

Fedora (antigamente chamado Fedora Core) é uma distribuição Linux baseada em pacotes RPM, criada pela Red Hat. Atualmente mantida pelo Projeto Fedora (Fedora Project).

Principais distribuições Linux

  • Ubuntu
  • Debian
  • Fedora
